== Description ==

This plugin allows you to easily add the "theme-color" meta tag to your website.

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload `wessels-browser-theme` directory to `/wp-content/plugins/` in the WordPress directory.
2. Activate the plugin with the 'Plugins' menu.
3. Click on the 'Plugins' menu on 'Settings' or go to (Settings -> Browser Theme Color)
4. Type your color
